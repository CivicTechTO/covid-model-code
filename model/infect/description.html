    • General Controls
      • Run/Stop
        • This toggles between running and stopped.
      • Exposition/Game
        • Toggles between exposition mode and game mode. May only be changed before you start the model running.
      • Speed
        • Toggles between displaying results after every model step and displaying results after a number of model steps.
      • Step size
        • Toggles between a small step size in the model and a large one. May only be changed before you start the model running.

      • Mask level
